# Project Memory — 保险独立比价平台

## 核心定位
只开方不卖药：保险产品客观对比 + 需求测评 + 理赔协助，零佣金零导购

## 商业模式（组合）
- 对比/测评：免费引流
- 订阅（高级报告/保单体检/续保提醒）：¥9.9-29/月
- 理赔协助（按次）：¥99-499/单 ← 核心利润
- B2B SaaS（经纪人账号）：¥99-299/月/号

## 当前状态
创始人正在准备保险代理人考试，计划路径：考证 → 入职经纪公司6-12个月积累经验 → 再出来做独立平台

## 关键认知
- 全球无成熟先例（比价+理赔+零佣金模式是蓝海空白）
- 国内现有平台（深蓝保/慧择/蚂蚁保全选）均以佣金为收入，非竞品
- 最接近参照：美国ACAP（独立理赔协助师，但无技术平台）
- 建议"先手动后自动"：用微信群接单手动出报告，验证需求后再开发
- 用AI辅助条款结构化、内容产出、理赔SOP整理

## MVP分期
- P0: 50款产品库 + 对比器 + 需求测评（工具）
- P1: 条款NLP + 保费试算 + 报告生成 + 用户账户
- P2: 理赔助手 + B2B API + 付费墙
